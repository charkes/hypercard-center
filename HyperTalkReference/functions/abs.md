# abs

<code><pre>
the abs of [ph:number]
</pre></code>

Value returned:  number equal to the absolute value of<code> [ph:number</code>] 


## Examples

```
the abs of -7
the abs of -7 + 3 -- returns 10
the abs of (-7 + 3) -- returns 4
```
