# tool

```
the tool
```

Value returned: a text string equal to the name of the currently chosen tool (that is, the `[ph:toolName]`)

(Click `[ph:toolName]` for a list of all the tool names.)

## Examples

```
if the tool is not "browse tool" then ...
```
