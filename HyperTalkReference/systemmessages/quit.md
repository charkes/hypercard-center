# quit

<code><pre>
quit
</pre></code>

Handler:

<code><pre>
on quit
  [ph:statements]
end quit
</pre></code>

HyperCard sends the <code>quit</code> message to the current card when the user chooses Quit HyperCard from the File menu (or presses Command-Q), just before HyperCard quits.
