# moveWindow

```
moveWindow
```

Handler:

```
on moveWindow
  [ph:statements]
end moveWindow
```

HyperCard sends the `moveWindow` message to the current card when the user or a script moves the window. In many cases, HyperCard will send a `sizeWindow` message immediately before a `moveWindow` message.
