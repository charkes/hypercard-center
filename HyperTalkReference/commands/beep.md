# beep

<code><pre>
beep
beep <i>posInteger</i>
</pre></code>


The<code> beep </code>command sounds the Macintosh system beep.

If you specify a<code> <i>posInteger</i>,</code> your Macintosh beeps that many times.

## Demo Script

<code><pre>
<code><pre>
on mouseUp
 <b>beep</b> 2
 answer "Beep! Beep!"
end mouseUp
</pre></code>
</pre></code>
