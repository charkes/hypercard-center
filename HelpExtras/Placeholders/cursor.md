### cursor

An expression, used to set the <code>cursor</code> property in a script, that evaluates to one of these names:

<code><pre>
Ibeam
cross
plus
watch
hand
arrow
busy
none
</pre></code>

Additionally, the first four names can be referred to by number:

<code><pre>
Ibeam      1
cross      2
plus       3
watch      4
</pre></code>
