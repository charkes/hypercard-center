### text

An expression that evaluates to a text string. 

A text expression can be a literal string surrounded by quotation marks or the value of a container.

For example: 

<code><pre>
"Hello"
"The sum is:" && 3
card field 1
line 1 of bg fld "Address"
the Message box
the selection
myVar -- a variable
</pre></code>
